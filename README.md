# Nebula Web Console

## QuickStart

### Dev

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Deploy

```bash
$ npm run build - build
$ npm start - start server
```

### Relative Commands

- Use `npm run lint` to check code style
- Use `npm test` to run unit test
- se `npm run clean` to clean compiled js at development mode once

### Required

- Node.js 8.x
- Typescript 2.8+
